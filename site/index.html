How It Works

Review your AI agent's output in five steps — using agent skills that automate the entire loop.

01

Your AI agent writes code

Ask it to build a feature, fix a bug, or refactor. It produces files you need to review.

02

Open in mdownreview

Browse the file tree, read rendered markdown, and leave inline review comments.

Files
📄 validate.ts 1
📄 middleware.ts
validate.ts
1export function validate(token: string) {
2 return verify(token, SECRET);
You issue
Missing null check — token could be undefined
3}
03

Summarize with /mdownreview:read

Your agent scans all .review.yaml sidecars and lists every unresolved comment.

04

Fix with /mdownreview:review

Your agent reads comments, makes fixes, and marks them resolved.

05

Clean up with /mdownreview:cleanup

Removes .review.yaml sidecar files once all comments are resolved.

Agent Skills

Install the agent skills to let your AI agent read and act on review comments.

/plugin marketplace add dryotta/mdownreview-skills
/plugin install mdownreview@mdownreview-skills
SkillDescription
openFind, install, and launch the mdownreview desktop app
readScan for review sidecars and list unresolved comments
reviewOrchestrate the full cycle — read, fix, and clean up
cleanupDelete sidecar files where all comments are resolved

Updating

Keep the app and skills up to date.

🔄 App

mdownreview checks for updates automatically on launch and installs them in the background. No action needed.

🔌 Skills

Update the agent skills to get the latest features:

/plugin update mdownreview-skills
