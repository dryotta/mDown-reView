# Persona seeds

Each `*.md` file is a brief: 5-15 lines describing how a particular kind of
user behaves and what they tend to expose. The agent reads one or more
briefs at the start of a run and uses them as **inspiration**, not a script.

The agent is expected to:
- Read at least one brief.
- Improvise within the brief's spirit.
- Mix in unscripted "what if" actions to find rare states.
- Stop early if the same screen recurs without new findings.
